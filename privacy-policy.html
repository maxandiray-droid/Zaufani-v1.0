🔒 Politique de Confidentialité

Dernière mise à jour: 26 avril 2026

1. Introduction

Zaufani Rodzina ("l'Application") est une Progressive Web App (PWA) gratuite et open-source conçue pour faciliter la communication et les échanges au sein des familles.

Commitment: Votre vie privée est importante. Cette application est conçue avec la confidentialité par défaut.

2. Collecte de Données

Données que nous collectons:

  • Données de profil: Nom, email, avatar (stockés localement)
  • Données de marketplace: Listings, transactions, avis
  • Données de localisation: Coordonnées GPS (optionnel, activé par l'utilisateur)
  • Messages: Conversations entre utilisateurs
  • Paiements: Transactions via PayPal (PayPal gère ces données)

Données que nous NE collectons PAS:

  • ❌ Historique de navigation
  • ❌ Contacts de votre téléphone
  • ❌ Photos ou fichiers non partagés par vous
  • ❌ Métadonnées de localisation en arrière-plan
  • ❌ Données comportementales pour publicités
  • ❌ Informations biométriques

3. Stockage des Données

Actuellement (v1.0): Les données sont stockées en localStorage de votre appareil.

  • Accessibles UNIQUEMENT à vous
  • Non synchronisées avec nos serveurs
  • Effacées si vous videz le cache du navigateur

Post-MVP (v2.0): Migration vers Firebase Realtime Database avec:

  • Chiffrement end-to-end optionnel
  • Synchronisation sécurisée entre appareils
  • Backup automatique

4. Partage de Données

Nous ne partageons JAMAIS vos données avec:

  • ❌ Annonceurs ou sociétés de marketing
  • ❌ Gouvernements ou tiers (sauf si légalement requis)
  • ❌ Sociétés data brokers

Exceptions légales: Nous pouvons être tenus de révéler des informations pour:

  • Conformité légale
  • Enquêtes criminelles (avec mandat)
  • Protection de droits légaux

5. Localisation GPS

Fonctionnement:

  • Activé par l'utilisateur uniquement
  • Utilise l'API HTML5 Geolocation
  • Coordonnées JAMAIS envoyées aux serveurs
  • Affichées uniquement sur la carte Leaflet locale
  • Utilisateurs de la même famille voient les positions respectifs

Contrôle: Vous pouvez désactiver à tout moment via le bouton "📍 Partager ma localisation"

6. PayPal & Paiements

Les paiements marketplace sont traités par PayPal.

  • Nous ne voyons JAMAIS vos numéros de carte
  • PayPal gère toutes les données de paiement
  • Nous ne stockons que l'ID de transaction

Consultez: Politique PayPal

7. Google AdSense

Actuellement configuré mais désactivé. Quand activé:

  • Annonces basées uniquement sur le contenu de la page
  • Pas de suivi comportemental inter-site
  • Respect des préférences utilisateur GDPR

8. Cookies

L'application utilise UNIQUEMENT:

  • localStorage pour les préférences utilisateur
  • Pas de cookies de suivi tiers
  • Pas de cookies publicitaires

9. Sécurité

Mesures de sécurité implémentées:

  • ✓ HTTPS obligatoire
  • ✓ Pas de données en texte clair
  • ✓ Service Worker pour protection offline
  • ✓ Validation input pour XSS/injection
  • ✓ CORS headers configurés

À implémenter (v2.0):

  • Hachage SHA-256 des mots de passe
  • Chiffrement des messages sensibles
  • 2FA optionnel

10. Droits Utilisateurs (RGPD/CCPA)

Vous avez le droit à:

  • ✓ Accès: Voir toutes vos données (localStorage export)
  • ✓ Rectification: Modifier vos informations personnelles
  • ✓ Suppression: Demander la suppression de compte
  • ✓ Portabilité: Exporter vos données
  • ✓ Opposition: Refuser certains traitements

Pour exercer ces droits: support@zaufani-rodzina.com

11. Enfants & Mineurs

Cette application est conçue pour les familles (tous âges).

  • Les enfants < 13 ans doivent avoir le consentement parental
  • Contenu adapté à tous les âges
  • Pas de contenus dangereux ou frauduleux tolérés

12. Modifications de cette Politique

Nous pouvons mettre à jour cette politique. Les modifications majeures seront communiquées via:

  • Email aux utilisateurs
  • Notification in-app
  • Publication sur cette page
