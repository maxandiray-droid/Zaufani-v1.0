  2. Open project folder: zaufani-app/

  4. Select APK
  5. Choose keystore: ../zaufani-signing-key.keystore
  6. Password: temp123456
  7. Key alias: zaufani
  8. Build type: Release
  9. Click Build → APK ready in app/release/

  Estimated time: 15-20 minutes

  ─────────────────────────────────────────────────────

  OPTION B - Using EAS (Cloud-based, Fastest)
  ─────────────────────────────────────────────────────
  1. Create account at eas.expo.dev
  2. Run: npm install -g eas-cli
  3. Run: eas build --platform android --type apk
  4. Download signed APK from EAS
  5. Done!

  Estimated time: 10-15 minutes

  ─────────────────────────────────────────────────────

  OPTION C - Command Line (Advanced)
  ─────────────────────────────────────────────────────

  export ANDROID_HOME=/path/to/android/sdk
  cd zaufani-app
  cordova build android --release
  # APK in: platforms/android/app/build/outputs/apk/release/
